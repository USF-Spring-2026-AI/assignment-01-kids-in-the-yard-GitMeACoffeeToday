See assignment details on Canvas.


# Repository Directory Guide
- family_tree.py 
    - Contains classes and functions relevant to creating and iterating through
      the family tree, dependent upon imports from data_frame_handler.py and person.py.

- data_frame_handler.py
    - Contains DataFrameHandler class which is responsible for reading and
      processing all information from the .csv files, (e.g. name, age, etc).

- person.py
    - Contains the Person class, of which the family tree is made up of.
      Dependent on imports from data_frame_handler.py to populate variables within each person node.

- main.py
    - Contains the implementation for user interactivity, currently supports the
      (3) required interactions as specified in the assignment details.


- comparison
  - Contains the LLM generated code, the responses (responses.txt) to the given questions on the assignment, and the prompt used (prompt.txt).
